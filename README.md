# Civitai 下载解析工具（使用说明）

本软件用于解析 Civitai / HuggingFace 模型链接，批量加入下载队列并管理下载进度，自动断点续传，重新申请下载地址，自动获取封面缩略图，支持哈希校验与本地模型整理与查看，生成离线HTML页面方便查看模型详情，带提示词解析和工作流解析，一键复制参考。

## 运行

- 直接运行：`CivitaiDownloader.exe`
- 默认下载目录：`程序目录\downloads\models`
- 默认封面缓存：`程序目录\downloads\images`

## 配置与保存位置

软件的配置保存在 `user_config.json`，用于记住下载目录、代理、并发数、API Key 等。

- 默认（推荐）：`%APPDATA%\CivitaiDownloader\user_config.json`（或 `%LOCALAPPDATA%\CivitaiDownloader\user_config.json`）
- 便携模式：设置环境变量 `CIVITAI_PORTABLE_CONFIG=1`，配置保存到 exe 同目录

## 主界面说明

软件包含三个标签页：

- 批量下载：粘贴链接并解析，选择文件后加入下载队列
- 下载管理：查看下载状态、暂停/继续、删除任务、哈希校验、查看已完成
- 模型管理（需解锁）：扫描本地模型目录与信息文件，根据哈希在C站搜寻对应模型，如果存在补全模型详细信息和封面，并可根据模型信息整理和分类目录，还能生成本地离线HTML页面方便查看模型示例。
- 离线HTML（需解锁）：查看本地离线关于模型的HTML页面，含模型详细描述和示例图，为方便在没有可靠网络环境下查看模型示例。且内置提示词解析工具，可以解析提示词，工作流。带一键复制，可直接套用。

## 批量下载（解析链接）

1. 在输入框粘贴多行内容（每行一个）。可直接批量解析下载链接，并添加到下载队列。
2. 支持的常见输入：
   - Civitai 模型页链接（包含 modelVersionId）
   - Civitai 版本链接
   - Civitai API 下载链接
3. 点击解析后会生成预览条目。
4. 勾选需要的条目，点击“加入下载”，任务会进入“下载管理”。

预览条目通常会显示模型名称/类型、版本信息、文件参数（精度 fp、格式 format、尺寸 size、类型 type）。这些参数用于区分同版本下不同文件，避免误选或误判重复。

## 下载管理（队列与状态）

### 状态

- 下载中：等待中 / 等待下载 / 下载中 / 暂停 / 校验中 / 失败
- 已完成：已下载完成的任务（含本地文件路径、封面与相关信息）

### 常用操作

- 开始/暂停：控制单个任务或队列执行
- 删除：从队列移除任务（不一定删除本地文件，取决于界面提示）
- 重试：失败任务可自动重新开始，会用API请求新的地址根据分片断点续传。（但是也有意外，可能遇到从头开始下载的情况。目录会将前一份downloading文件备份为`downloading_时间戳backup`，方便你查看与恢复。可是大概率续传下完哈希校验不会通过，需要手动重试。）

### 重复文件处理

当目标文件在下载目录中已存在时，会提示是否重新下载。对于同一模型不同精度/格式的文件，软件会依据文件参数生成不同文件名并进行匹配，降低误判概率。

### 哈希校验

下载完成后可进行哈希校验：

- 校验中：任务仍显示在“下载中”
- 校验失败：任务不会进入“已完成”，方便你直接重试或处理。

## 封面缩略图来源

缩略图显示优先级：

1. 已绑定本地封面路径且文件存在：直接加载
2. 模型文件存在：尝试加载同名图片文件（例如 `xxx.safetensors` 对应 `xxx.jpg/png/webp`）
3. 仍没有：根据版本的封面媒体 URL 下载到本地后显示，如果封面媒体是视频mp4，会提取第一帧作为封面。

重新打开软件时，会优先从本地封面恢复显示。

## 常见设置建议

- API Key：必填，用于访问 Civitai API，C站模型下载需要带API的请求头
- 代理：在网络受限环境下建议开启并设置代理地址
- 最大任务数：并发过高可能触发限流或导致失败率上升，可按网络情况调整
- 下载目录：建议设置到你常用的模型目录，便于后续管理

## 常见问题排查

- 下载失败/403：检查网络、代理与 API Key，适当降低并发并重试
