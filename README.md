# 🧸 Civitai 下载解析工具｜超全使用指南
AI 模型下载/管理/整理 一站式神器✨

本软件专门用来解析 Civitai / HuggingFace 模型链接
批量下载➕断点续传➕自动封面➕哈希校验
本地模型整理 + 离线HTML查看 + 提示词/工作流解析
一键复制参考，新手也能轻松上手～

---

## 🚀 如何运行
- 直接打开：`CivitaiDownloader.exe`
- 默认模型下载：`程序目录\downloads\models`
- 默认封面缓存：`程序目录\downloads\images`

---

## ⚙️ 配置文件位置
配置存在 `user_config.json`
记住下载路径、代理、并发数、API Key 等

- 推荐默认：`%APPDATA%\CivitaiDownloader\user_config.json`
- 便携模式：设置环境变量 `CIVITAI_PORTABLE_CONFIG=1`

---

## 📌 主界面 4 大功能
✅ 批量下载：粘贴链接 → 自动解析 → 一键排队

## 📥 批量下载教程
1. 粘贴多行链接（每行一个）
2. 支持：Civitai模型页 / 版本页 / API下载链接
3. 点【解析】→ 勾选文件 → 【加入下载】
自动识别：模型名、版本、精度、格式、大小

---

## 📊 下载管理功能
▪ 下载中：等待/下载/暂停/校验/失败
▪ 已完成：查看文件路径+封面+信息
▪ 重试：自动重新获取地址 + 断点续传
▪ 哈希校验：确保模型完整不损坏

---

## 🖼️ 封面自动获取规则
1. 本地已有封面 → 优先显示
2. 模型同名图片（png/jpg/webp）→ 自动加载
3. 网络下载封面 / 视频提取第一帧
重启软件也能快速显示封面图～

---

## 🔧 必看设置建议
- API Key：必填！用于 Civitai 下载权限
- 代理：网络不好一定要开
- 最大任务数：太高容易失败，适中即可
- 下载目录：直接设为 ComfyUI 模型目录更方便

---

## ❌ 常见问题
下载失败 / 403？
检查：网络 → 代理 → API Key → 降低并发重试
